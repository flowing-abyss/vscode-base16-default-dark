# Base16 Default Dark

A clean, minimal dark theme for VS Code, ported from the [Base16](https://github.com/chriskempson/base16) color scheme.

## Install

Search **"Base16 Default Dark"** in the VS Code Extensions panel, or install from a `.vsix` file via the command palette (`Extensions: Install from VSIX`).
