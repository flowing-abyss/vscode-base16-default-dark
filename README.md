# Base16 Default Dark

A dark theme for VS Code, based on the [Base16](https://github.com/chriskempson/base16) color scheme.

## Install

Download the `.vsix` from the [Releases](https://github.com/flowing-abyss/vscode-base16-default-dark/releases) page, then run **Extensions: Install from VSIX...** from the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`).
