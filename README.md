# Base16 Default Dark

A dark theme for VS Code:, based on the [Base16](https://github.com/chriskempson/base16) color scheme.

## Install

Open the Extensions panel (`Ctrl+Shift+X` / `Cmd+Shift+X`), search for **"Base16 Default Dark"**, and click Install.

Alternatively, download the `.vsix` from the [Releases](https://github.com/flowing-abyss/vscode-base16-default-dark/releases) page, then run **Extensions: Install from VSIX...** from the Command Palette (`Ctrl+Shift+P` / `Cmd+Shift+P`).
